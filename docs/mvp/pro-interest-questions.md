# Pro Interest Questions

Use these questions to test whether Freemium + Pro has a real paid upgrade path.

## User Context

1. Do you currently use Stripe subscriptions?
2. Roughly what MRR range are you in?
3. Are you primarily working on a Mac?
4. How often do you check MRR today?

## Free Alpha Reaction

1. Did seeing MRR on your Lock Screen feel useful after the first day?
2. Would you keep it enabled for a full week?
3. What felt most valuable: motivation, convenience, aesthetics, privacy, or something else?
4. What felt risky or annoying?

## Pro Pull

Which would you pay for?

- Custom lock-screen design.
- Position / size / typography controls.
- Refresh frequency and manual refresh.
- Multi-currency display polish.
- MRR goal / milestone / delta.
- Customer count or active subscription count.
- Signed / notarized installer.
- Ongoing macOS compatibility updates.

## Pricing Signal

1. Would you prefer one-time purchase, monthly subscription, annual subscription, or lifetime license?
2. What price would feel obviously reasonable?
3. What price would make you stop and think?
4. What feature would need to exist before you would pay?

## Decision Question

If this stayed focused only on being a beautiful daily MRR signal, would you still want it, or would you need full analytics?
