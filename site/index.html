What it does

One job: make MRR visible.

Reads Stripe locally

The app fetches subscription data from Stripe directly on your Mac and computes Stripe-like MRR locally.

Lives on the Lock Screen

The overlay appears as a glass panel in the place you already see before starting work.

Stays intentionally narrow

This is not a full analytics dashboard, finance tool, or Stripe Dashboard replacement.

Local-first security

No alpha server stores your Stripe key or MRR.

Restricted key Use a read-only Stripe restricted API key, not a full-access secret key.
Keychain storage The Stripe key is stored in macOS Keychain under your user account.
Local cache Only the last-good MRR value and timestamp are cached locally for stale/offline display.

Free and Pro boundary

Free stays useful. Pro is convenience.

Free source alpha

Build from source, configure a restricted Stripe key locally, show MRR on the Lock Screen, keep last-good cache, and adjust refresh and position settings.

Possible Pro

Signed installer, auto-update, more visual styles, multi-display controls, MRR goals, and compatibility maintenance for future macOS releases.

Not the plan

No hosted Stripe key storage, no full analytics dashboard, no customer/payment warehouse, and no replacement for Stripe Dashboard.
